ワンクリックインストール

5分以内で稼働開始できる効率的なセットアッププロセス

📦 方法1: インストールスクリプト

curl -fsSL https://raw.githubusercontent.com/NFTTechnology/NFTT-GitHub-Workflows/main/install.sh | bash

対話型インストーラーが環境をチェックし、最適な設定を提案します。

🔧 方法2: GitHub CLI Extension

# 拡張機能をインストール
gh extension install NFTTechnology/NFTT-GitHub-Workflows --force
# ワークフローをインストール
gh nftt-workflows install

GitHub CLIユーザー向けの統合されたインストール体験。

🔑 必要なAPIキー

インストール後、以下のAPIキーを設定してください：

  • ✓ ANTHROPIC_API_KEY - 取得する
  • ✓ OPENAI_API_KEY - 取得する
  • ✓ GEMINI_API_KEY - 取得する
